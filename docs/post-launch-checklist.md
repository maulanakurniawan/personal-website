# Post-Launch Checklist

## Stability & performance
- [ ] Review error rates and latency trends.
- [ ] Validate webhook delivery success rate.
- [ ] Address top support issues from launch week.
- [ ] Schedule follow-up load testing.

## Customer feedback
- [ ] Gather feedback from first users.
- [ ] Prioritize fixes and quick wins.
- [ ] Track feature requests in roadmap.

## Marketing follow-through
- [ ] Share customer testimonials or case studies.
- [ ] Engage commenters and questions on launch post.
- [ ] Update landing page with launch highlights.

## Retrospective
- [ ] Hold launch retrospective meeting.
- [ ] Document lessons learned.
- [ ] Update runbooks and checklists.
