# Marketing Plan

## Website submission & discovery
- [ ] Submit sitemap to Google Search Console.
- [ ] Verify site ownership and monitor indexing status.
- [ ] Submit to Bing Webmaster Tools.
- [ ] Ensure robots.txt allows key pages.
- [ ] Add structured data (Organization, Product, FAQ).

## Launch content
- [ ] Publish a launch blog post on the website.
- [ ] Draft a short product update for email subscribers.
- [ ] Prepare a press kit (logo, screenshots, boilerplate).

## Social & community
- [ ] Announce on X, LinkedIn, and relevant communities.
- [ ] Post a demo video or GIF for key workflows.
- [ ] Engage with comments and answer questions.

## Listings & directories
- [ ] Submit to Product Hunt (if ready for community launch).
- [ ] Submit to startup directories relevant to developers.
- [ ] Update GitHub README with launch info.

## Example launch post outline
1. **Headline**: What problem the product solves.
2. **Who it is for**: Target audience and use cases.
3. **Key features**: 3-5 highlights.
4. **How it works**: Simple flow with a screenshot or GIF.
5. **Call to action**: Link to sign up or schedule a demo.
