# Launch Checklist

## Final verification
- [ ] Run smoke tests on production.
- [ ] Validate critical monitoring alerts trigger correctly.
- [ ] Confirm billing and plan enforcement.
- [ ] Verify webhook delivery in production with test endpoint.

## Release execution
- [ ] Merge and deploy release tag.
- [ ] Announce launch internally.
- [ ] Update status page with launch note.
- [ ] Enable onboarding emails and in-app guidance.

## Marketing & outreach
- [ ] Publish launch post.
- [ ] Share announcement on social channels.
- [ ] Notify waitlist or beta users.
- [ ] Submit to relevant directories.

## Support readiness
- [ ] Staff support coverage for launch window.
- [ ] Prepare escalation paths for incidents.
- [ ] Set up live dashboard for key metrics.
